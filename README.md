# Web scraping API with FastAPI
<h1>Project Setup</h1>
<p> Make the directory project </P>
<p> Change to that directory : $cd dic_name</p>
<p> Create the virtual enviroments : $python -m venv .venv "You can you .venv any name"</p>
<p> Activate the venv : $source .venv/bin/activate " You can check using $which python3</p>
<p> Install FastAPI standard : $pip install "fastapi[standard]"</p>
<p> Install dependency request & beautifulsoup4 : $pip install request and beattifulsoup "for web scarping"</p>
<p> Add requirements.txt : $ pip freeze > requirements.txt "for denpencies list"</p>
<p> Create main.py : $touch main.py</p>
<p> Test run : $uvicorn main:app --reload<p>
